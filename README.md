# Spotify-dashboard
# 🎧 Spotify Analytics Dashboard – Power BI

An interactive **Spotify Analytics Dashboard** built using **Power BI** to explore songs, artists, popularity trends, and album insights.
This dashboard provides a clean UI inspired by Spotify with deep analytical insights for music data exploration.

---

# 📊 Dashboard Preview

## 🏠 Home

Landing page with Spotify-themed UI and navigation to all report pages.

## 📈 Overview

* Total Distinct Songs
* Count of Artists
* Average Duration
* Average Popularity
* Songs by Album Type
* Explicit vs Non-Explicit Distribution
* Avg Popularity by Month
* Distinct Songs by Month

## 🎤 Artists Analysis

* Artist by Songs
* Songs by Popularity
* Songs by Hit
* Artist-level metrics:

  * Avg Duration
  * Avg Popularity
  * Max / Min Popularity
  * Hit Ratio

## 🎵 Songs Analysis

* Song by Artist filter
* Songs by Popularity
* High Popularity Songs
* Song level table with:

  * Release Date
  * Album Type
  * Avg Duration
  * Popularity Metrics

# 📂 Dataset Details

The dataset includes:

* Song Name
* Artist
* Album Type
* Release Date
* Duration
* Popularity
* Explicit Flag
* Track Metrics

---

# 📊 Key Insights

* Most songs are released as **Albums vs Singles**
* Popularity varies significantly by **Artist**
* Monthly trend shows **declining average popularity** over time
* Few artists dominate **high popularity hits**
* Explicit tracks form a significant portion of total songs
